# hyundon.dev

This repo contains the source code for [my personal website](https://hyundon.dev)

## Built with

- Astro
- TypeScript
- Tailwind CSS
- Markdown

## Run locally

```bash
git clone https://github.com/hyundonmoon/hyundon.dev.git
cd hyundon.dev
pnpm i
pnpm run start
```
